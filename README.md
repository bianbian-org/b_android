## polyhome_b

### polyhomeb：博力云社区

### Android版本(B端)

- LightWidget：一些轻量级控件库
- PhotoPicker：照片选择库
- alipay：支付宝支付相关库
- app：项目主module
- corelib：项目核心层支持库，包含常用工具和部分框架
- doormaster：智果蓝牙门锁库
- lib2：网络层库框架
- loading-more-view：加载更多相关库
- magicindicator：viewpager的indicator支持库
- paycorelib：支付相关的链接库
- ptr_lib：HUD加载动画库
- processdialog：MD风格对话框兼容库
- slide2unlock：滑动解锁控件库
- slidngmenulib：侧滑库
- unionpay：银联支付支持库
- wechatpay：微信支付支持库
- zixing：二维码相关支持库
